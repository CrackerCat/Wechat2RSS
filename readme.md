# 微信公众号RSS

https://wechat2rss.xlab.app

## 目标

本项目从2021年9月启动，旨在提供**长期稳定可用**的微信公众号RSS服务

尽力确保文章更新周期在24小时内（从作者发布到RSS收录）

## 免费服务
